Steps for MNE analysis

1. Find dead channels (using checkForDeadChannels in Matlab)
2. Denoise sqd file, using sqddenoise package
3. Convert sqd file to txt using matsqd_txt.m  -->NOTE: this takes a very long time!
    e.g. matsqd_txt('AUDI','R0253','AUDI_blocked-Filtered')
    
4. Make a coreg.txt file in MEG160, following directions on MEGwiki for Sqd to Fif conversion
	e.g. KIT/R0253_Narwhal_coreg.txt
    **NOTE: Elizabeth will come back and modify details here so you don't have to go to the MEGwiki
5. Convert txt file to fif using sqdtxt2fif.csh, 
    e.g. ./sqdtxt2fif.csh AUDI R0253 AUDI_blocked 500
    
6. Mark bad channels using preProc_setup.csh
    e.g. ./preProc_setup.csh AUDI R1841 
    
7. Convert eve file from binary using binary_conversion.py
    e.g. python binary_conversion.py AUDI R0253 AUDI_random
8. Find the samples containing blinks using rej_fif2blinkrej.m
    e.g. rej_fif2blinkrej('AUDI','R0253','AUDI_random',0);
9. Make a new eve file with the rejected trials using rej_rej2eve.py
    e.g. python rej_rej2eve.py AUDI R0253 

10. Make .ave files, either manually or automatically
   --automatically: python /Users/Shared/MNE_scripts/makeAveFiles.py AUDI R1841 M100
   --manually: if you don't want the default conditions listed in condCodes.py, look at an existing .ave file for guidance on the format. Save file into file named data/R1841/ave/R1841_M100.ave
   **note that running automatic route will overwrite any manually created files with same name

11. Do averaging with preProc_avg.csh
    e.g. ./preProc_avg.csh NARWHAL R1855 Narwhal
